# nvim-config

## Requirement
- Ruby
  - neovim
  - solargraph
  - rubocop
- Python
  - neovim
  - python-language-server
  - flake8
- node.js
  - neovim
  - javascript-typescript-langserver
  - standard

## Usage
```sh
echo 'export XDG_CONFIG_HOME=~/.config' >> ~/.bashrc
source ~/.bashrc
mkdir ~/.config
git clone https://github.com/hotaru51/nvim-config.git ~/.config/nvim
```
